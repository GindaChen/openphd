Get started in 2 minutes

Manual setup if you prefer the step-by-step approach

1

Clone the repo

git clone https://github.com/GindaChen/openphd.git && cd openphd
2

Install dependencies

cd services/kanban && npm install && cd ../../frontend && npm install
3

Start the app

This runs both the API server and the Vite dev server.

npm run dev

Enable AI (optional)

Set your Anthropic API key to unlock the AI agent. You can also configure this in Settings.

ANTHROPIC_API_KEY=sk-ant-... npm run dev
